Detailed Description

Custom SkinLoader class for our custom TTFSkin class to create bitmap fonts on runtime according to the density of the display.

Code is very similar to the original SkinLoader code except that getDependencies(String, FileHandle, SkinParameter) also includes bitmap font dependencies according to the given SkinParameter to load the bitmap fonts before loading the skin. Also loadSync(AssetManager, String, FileHandle, SkinParameter) is creating our own TTFSkin instance.
