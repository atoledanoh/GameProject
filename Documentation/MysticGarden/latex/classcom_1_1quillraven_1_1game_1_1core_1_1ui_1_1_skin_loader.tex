\doxysubsection{Detailed Description}
Custom \mbox{\hyperlink{classcom_1_1quillraven_1_1game_1_1core_1_1ui_1_1_skin_loader}{Skin\+Loader}} class for our custom \mbox{\hyperlink{classcom_1_1quillraven_1_1game_1_1core_1_1ui_1_1_t_t_f_skin}{TTFSkin}} class to create bitmap fonts on runtime according to the density of the display. 

Code is very similar to the original \mbox{\hyperlink{classcom_1_1quillraven_1_1game_1_1core_1_1ui_1_1_skin_loader}{Skin\+Loader}} code except that \mbox{\hyperlink{classcom_1_1quillraven_1_1game_1_1core_1_1ui_1_1_skin_loader_ae10957cba433a67dc0ba523ac72336ad}{get\+Dependencies(\+String, File\+Handle, Skin\+Parameter)}} also includes bitmap font dependencies according to the given \mbox{\hyperlink{}{Skin\+Parameter}} to load the bitmap fonts before loading the skin. Also \mbox{\hyperlink{classcom_1_1quillraven_1_1game_1_1core_1_1ui_1_1_skin_loader_a22772193d5ef410a244adc10b661ba1b}{load\+Sync(\+Asset\+Manager, String, File\+Handle, Skin\+Parameter)}} is creating our own \mbox{\hyperlink{classcom_1_1quillraven_1_1game_1_1core_1_1ui_1_1_t_t_f_skin}{TTFSkin}} instance.
