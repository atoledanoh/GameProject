\doxysubsection{Detailed Description}
Custom skin class to avoid the \char`\"{}\+Pixmap already disposed\char`\"{} exception when disposing the skin. The reason is that the bitmap fonts that are created on runtime are not handled correctly within the \mbox{\hyperlink{}{com.\+badlogic.\+gdx.\+assets.\+Asset\+Manager}} and get therefore disposed twice.
